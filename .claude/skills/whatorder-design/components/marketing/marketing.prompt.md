Marketing-site building blocks for the dark surface. `SectionLabel` is the green eyebrow, `StepCard` is a numbered feature card, `QuoteBlock` is a testimonial panel.

```jsx
<SectionLabel>How it works</SectionLabel>
<StepCard step="Step 02" title="Bot handles everything">
  Menu, items, quantities, extras — all through interactive messages.
</StepCard>
<QuoteBlock
  quote="My customers are already on WhatsApp. Now they order from me directly."
  author="— Owner, Vienna (pilot phase)"
/>
```
