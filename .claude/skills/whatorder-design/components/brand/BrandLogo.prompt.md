The WhatOrder brand lockup — rounded green mark plus the "WhatOrder" wordmark (with "Order" in green). Use it in nav bars, login cards, headers, and hero units.

```jsx
<BrandLogo size="md" variant="light" />
<BrandLogo size="lg" variant="dark" glow showWordmark={false} />
```

- `variant="light"` for light surfaces (dashboard), `variant="dark"` for the dark marketing site.
- `glow` adds the green hero halo; use only on dark backgrounds.
- Set `showWordmark={false}` for a mark-only avatar/favicon use.
